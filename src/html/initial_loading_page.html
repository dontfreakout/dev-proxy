Configure containers for proxy

Configure containers in your docker-compose.yml file by adding hostname, port, and protocol to environment variables: 

services:
          my-service:
            networks:
              proxy:
            environment:
              VIRTUAL_HOST: example.localhost
              VIRTUAL_PORT: 80 # port of the service
              VIRTUAL_PROTO: http # (valid options are https or http)

For multiple domains on one service, separate with commas: VIRTUAL_HOST: example.localhost,example2.localhost

Network Configuration

Add your service and container to the proxy network: 

networks:
          proxy:
            name: proxy_network # match the network name in the script
